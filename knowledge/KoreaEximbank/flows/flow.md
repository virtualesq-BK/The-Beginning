# Korea Eximbank — Flow (영문국제계약양식)

1. 업무안내 → 기업지원서비스 → 영문국제계약양식 진입  
2. 『영문국제계약해설』 소개 확인  
3. **국제계약서식 다운로드** 탭 (`HPHKWG048M01#tab2`) 이동  
4. 분류·상세분류·키워드로 필터/검색 (총 77건, 8페이지)  
5. 양식 DOC 다운로드  
6. 거래 조건에 맞게 문안 수정 후 내부·전문가 검토  

The Beginning 연동 가정:
1. 사용자 계약서 업로드  
2. AI가 조항 유형·리스크 요약  
3. Eximbank 표준 양식 분류(특히 PF·대출·건설·보증)로 협상 체크리스트 보강  
4. 고위험 시 전문가 연결
