# Lex-In — 톤앤매너 가이드

## 브랜드 보이스
전문적 · 조용함 · 따뜻함
과장하거나 소란스럽지 않게, 신뢰감 있는 전문가의 어조로 안심시키는 톤을 유지한다.

## 근거 수치 규칙
아래 실측(또는 사례 기반) 수치만 인용 가능. 새로운 수치는 검증 전까지 사용하지 않는다.
- 검토 시간: 2주 → 5분
- 리스크 파악 비용: 80만원 → 3만원

## 금지 표현 (우리사업_핵심정보_5줄.md 금기선과 연동, 확인 필요)
- 리스크를 "완전히 제거·100% 보장"한다는 표현
- 법률 자문을 대체한다는 문구 (당사는 법률 자문 제공자가 아님을 항상 명시)
- 고객 계약서 원문을 무단으로 인용·노출하는 것
- 특정 소송 결과나 승소를 보장하는 문구
